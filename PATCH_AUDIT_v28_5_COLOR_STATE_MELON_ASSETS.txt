Patch v2.28.5 - Color State Melon Assets

คำตอบเรื่องรูปแตงโมเปลี่ยนตามสี:
- เวอร์ชันก่อนหน้าใช้รูปจริงไฟล์เดียว + CSS filter
- รอบนี้เปลี่ยนเป็นรูปจริงแยกตามระดับการสุกจริง ๆ

เพิ่มรูปผลลัพธ์ 5 สถานะ:
A = อาจเน่า = เทาเข้ม
B = สุกเกิน = เทาอ่อน
C = สุกพอดี = แดง
D = ใกล้สุก = ส้ม/เหลือง
E = ดิบ = เขียวอ่อน/ขาวนวล

การทำงาน:
- เพิ่ม MELON_RESULT_IMAGES_V2285 เป็น data URI ใน index.html
- เพิ่ม function setResultMelonImage(group)
- ตอนวิเคราะห์เสร็จ จะเรียก setResultMelonImage(group)
- resetAll() จะตั้งกลับเป็น C/สุกพอดีเป็น default
- ยกเลิกการพึ่ง CSS filter เป็นหลัก เพื่อให้สีของรูปตรงและชัดกว่าเดิม
